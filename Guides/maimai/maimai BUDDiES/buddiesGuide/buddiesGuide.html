About

Welcome to the guide on installing and setting up maimai DX BUDDiES on your PC.

Prerequisites and Important Information

  • WinRAR - Required to extract the game files.
  • (Optional) Notepad++ - Recommended for editing configuration files.
  • Disclaimer: Do not distribute this software publicly due to potential legal issues.
  • Installation on a maimai FiNALE Cabinet is possible but not recommended.
  • COM port setup instructions are discussed separately for VR maimai and YuanCon ADX Controllers.

Installation

  1. Extract the archive using WinRAR or 7zip.
  2. Upon opening the first few folders, you'll be greeted with 4 more folders and 1 zip file. Open the folder labelled "app".
  3. Next, open the folder labelled "package". Normally you'll need to install segatools (Software that allows the game to run) with a separate download but in this case it'll be preinstalled for you.
  4. Locate a file called "segatools.ini" and open it in any text editing software like notepad.
  5. In the first few lines, you'll need to route a few folders based on what the text asks for (amfs, options, and appdata).
    Example of what to copy and paste - "C:\[Replace this with wherever your game's folder is located]\maimaiBUDDiES\maimaiBUDs\option".
    Repeat with the other two folders - "C:\[Replace this with wherever your game's folder is located]\maimaiBUDDiES\maimaiBUDs\app\Package\appdata" and "C:\[Replace this with wherever your game's folder is located]\maimaiBUDDiES\maimaiBUDs\amfs"
  6. Save the edited file and locate another file called "start.bat". This will open the necessary software needed as well as the game itself.
    If there's a black screen, open CMD as an administrator and run the following command: "setx OPENSSL_ia32cap ~0x20000000"
    You'll be greeted with a few loading screens at first so wait for the game to finish initializing. Default controls are usually "QWEADZXC" for the buttons, 1/2 for Service and Test Mode, and 3 to insert credits if applicable. Player 2's controls are via the numpad. A mouse can be used to control the sliders and tap notes.
  7. If everything goes well, you'll be at the menu. However, if you see a small border with text along the upper half of the circle, follow the next steps.
  8. Press 2 (F1 or Home key if 2 doesn't work) on your keyboard to access the service menu.
  9. The text will all be in Japanese but press C until you've reached the 7th option called (ゲーム設定). Press X to access its settings.
  10. (Press C to scroll down and X to change a setting. Del Key or F3 alternatively) There will be 3 columns. Ensure the first is ON, the second is OFF with the number below being 1, and lastly, the third column ON with the bottom number being 25%.
  11. Scroll to the bottom and head out of that section. Scroll to the bottom again and press X to startup the game. If done correctly you should be able to start playing!
  12. Finally, your game after a certain time may be locked and you cannot enter the game. This is due to an option on the game itself, and you will have to enter the debug menu. Press F1 to enter, and the C button to scroll. Select the option below the one that says "aime". Once you select that, go to where it has the numbers and select it until it has text without any numbers.
  13. Enjoy!
